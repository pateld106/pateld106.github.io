Hackathon 1: XSS Attacks and Defenses

This hackathon focused on Cross-Site Scripting (XSS) attacks and defenses, providing hands-on experience in both exploiting and securing web applications against XSS vulnerabilities. The tasks included:

Attacks:

  • Exploited reflected XSS vulnerabilities across 7 levels (0-6) on the provided web application.
  • Injected payloads to display my name using alert() for each level.
  • Guessed server-side code vulnerabilities for levels 0-6 to understand how the exploits worked.
  • Demonstrated successful attacks with full URL screenshots and payload inspections.

Defenses:

  • Secured echo.php from Lab 1 by adding input validation and output encoding using htmlentities().
  • Protected the front-end prototype from Lab 2 by identifying external input channels, validating incoming data, and encoding data before displaying it on the webpage.
  • Committed all secure code revisions to GitHub with detailed documentation.

This exercise enhanced my understanding of both exploiting and defending against XSS vulnerabilities, equipping me with practical skills to secure web applications effectively.
